About Templates

Templates define the default category structure for new households. When a new user registers, the default template is automatically applied to create their initial categories.

  • Default Template: The template that will be used for new households automatically
  • Active Templates: Only active templates can be used (inactive templates are hidden from selection)
  • Categories: Each template contains a set of income, expense, and savings categories

Tip: You can manage template categories in detail by clicking the "View/Edit" button, or use Django Admin for full control.
